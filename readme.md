### Installation

Install the dependencies:
```sh
yarn install
```
or
```sh
npm install
```

### Development Workflow
Start a live-reload development server:
```sh
yarn run dev
```
or
```sh
npm run dev
```

Generate a production build:
```sh
yarn run build
```
or
```sh
npm run build
```
